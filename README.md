## âœ¨ Features

### Number Analysis
- **Prime Number Check**: Verify if a number is prime
- **Palindrome Check**: Check if a number reads the same forwards and backwards
- **Armstrong Check**: Determine if a number is an Armstrong number (sum of cubes of digits equals the number)
- **Magic Number Check**: Identify magic numbers

### Pattern Generation
- **Pattern 1: Pyramid**: Creates a symmetric pyramid pattern
- **Pattern 2: Increasing Triangle**: Generates an ascending triangle
- **Pattern 3: Decreasing Triangle**: Generates a descending triangle
- **Pattern 4: Right-Aligned Triangle**: Creates a right-aligned triangle pattern

### Prerequisites


```bash

```

3. Install dependencies
